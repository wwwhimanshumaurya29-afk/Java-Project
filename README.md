# Java-Project
Java-based Notepad application with file handling, text editing, printing, and event-driven UI using AWT.
 📝 Notepad Clone (Java AWT)

A simple desktop-based Notepad application built using Java AWT.  
This project demonstrates core concepts of GUI development, event-driven programming, and file handling in Java.

---

🚀 Features

- 📂 Open and Save text files  
- ✂️ Cut, Copy, Paste functionality  
- 🔍 Zoom In / Zoom Out support  
- 🖨️ Print functionality using PrinterJob API  
- 🧾 Menu-driven user interface  

---

 🛠️ Technologies Used

- Java  
- AWT (Abstract Window Toolkit)  
- Event-Driven Programming  
- Java I/O (BufferedReader, BufferedWriter)
